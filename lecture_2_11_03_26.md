## 📊 ЧАСТЬ 1: Excel — Unit-экономика (45 мин)

### Задание 1.1 — Обзор сети (10 мин)
Для каждой из 4 кофеен посчитайте:
- Количество транзакций
- Общую выручку (`total`)
- Общую себестоимость (`total_cost`)
- **Валовую маржу** = (выручка − себестоимость) / выручка × 100%
- Средний чек

### Задание 1.2 — Unit-экономика продуктов (15 мин)
Для каждого продукта:
- Выручка, себестоимость, **маржа** (₽ и %)
- Количество продаж
- Маржинальность = (price − cost) / price × 100%

**Вопросы:**
- Какой продукт — самый маржинальный?
- Какой — самый популярный?
- Есть ли продукты, которые стоит убрать из меню?

### Задание 1.3 — Проблемная кофейня (10 мин)
- Сравните средние скидки по кофейням
- Какая кофейня даёт самые большие скидки?
- Посчитайте: сколько денег потеряно на скидках (по каждой точке)?
- Формула потерь: `=total / (1 - discount_pct/100) - total`

### Задание 1.4 — Анализ бариста (10 мин)
- Средний чек по каждому бариста
- Количество транзакций на бариста
- Кто «звезда»? Кто отстаёт?

### Подготовка (5 мин)
1. Откройте **datalens.yandex.ru**
2. Войдите через Яндекс-аккаунт
3. Нажмите «Создать подключение» → «Файл» → загрузите `coffemania-sales.csv`
4. Создайте «Датасет» на основе подключения

### Задание 2.1 — KPI-карточки (10 мин)
Создайте 4 карточки 
- **Общая выручка** (поле `total`, агрегация: сумма)
- **Средний чек** (поле `total`, агрегация: среднее)
- **Количество заказов** (поле `transaction_id`, агрегация: количество)
- **Средняя маржа %** (создайте вычисляемое поле или используйте формулу)

### Задание 2.2 — Графики (15 мин)
Постройте 3-4 графика:

1. **Линейный:** Выручка по месяцам (ось X = month, Y = SUM(total))
2. **Столбчатый:** Выручка по кофейням (ось X = cafe, Y = SUM(total))
3. **Круговой:** Доля категорий в выручке (Кофе / Напитки / Еда)
4. **Тепловая карта:** Заказы по дням недели и часам

### Задание 2.3 — Сборка дашборда (10 мин)
1. Создайте «Дашборд»
2. Разместите карточки наверху
3. Графики — по центру
4. Добавьте **фильтр** по кофейне (селектор)
5. Дайте дашборду название: «КофеМания: CEO Dashboard»
