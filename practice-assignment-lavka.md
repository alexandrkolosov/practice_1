# 🛒 Практическое задание: «Аналитик Яндекс.Лавки»
## Мир данных вокруг нас | 1.5 часа

**Курс:** Business Intelligence для менеджеров
**Лекция 1:** Мир данных вокруг нас
**Формат:** Групповая работа (3-4 человека)
**Инструмент:** Excel или Google Sheets

---

## 📖 Легенда

Вы — команда аналитиков, нанятая руководством **Яндекс.Лавки**. Перед вами данные одной дарксторы (мини-склада) за 3 месяца.

**Ситуация:**
- Даркстора работает с января 2025 года
- Количество заказов растёт
- Но **оценки клиентов падают**, и руководство думает о закрытии точки
- Ваша задача: **найти причину и предложить решения**

---

## 📦 Описание датасета

Файл: `lavka-dataset.csv` (~4200 заказов, январь–март 2025)

| Поле | Описание | Пример |
|------|----------|--------|
| `order_id` | Номер заказа | 50001 |
| `date` | Дата | 2025-01-15 |
| `hour` | Час заказа (0-23) | 19 |
| `day_of_week` | День недели | Среда |
| `category` | Категория товара | Готовая еда |
| `items_count` | Кол-во товаров в заказе | 4 |
| `order_total` | Сумма заказа (₽) | 1200 |
| `delivery_time_min` | Время доставки (мин) | 18 |
| `customer_id` | ID клиента | U-0042 |
| `is_new_customer` | Новый клиент? (да/нет) | нет |
| `promo_used` | Промокод использован? (да/нет) | да |
| `rating` | Оценка 1-5 (пусто если не поставил) | 4 |
| `district` | Район доставки | Центральный |

## 🔎 Этап 1: Разведка данных (15 мин)

Ответьте на базовые вопросы:

- [ ] Сколько всего заказов в датасете?
- [ ] Какой период охватывают данные?
- [ ] Средний чек (и медиана!)
- [ ] Сколько уникальных клиентов?
- [ ] Какой % клиентов оставляет оценку?

**Подсказка:** Используйте функции `СЧЁТ`, `СРЕДНЕЕ`, `МЕДИАНА`, `СЧЁТЕСЛИ`, `УНИК` в Excel.

---

## 📊 Этап 2: Анализ (45 мин)

### Задание 2.1 — Динамика по месяцам
- Количество заказов, выручка, средняя оценка — **по каждому месяцу**
- Есть ли тренд? Что растёт, что падает?

### Задание 2.2 — Время доставки
- Среднее время доставки **по месяцам** — есть ли рост?
- Среднее время доставки **по районам**
- Как время доставки влияет на оценку?

### Задание 2.3 — Распределение по часам
- В какие часы больше всего заказов?
- В какие часы почти пусто? (Потенциал для роста!)
- Постройте график заказов по часам

### Задание 2.4 — Новые vs повторные клиенты
- Средний чек каждой группы
- Какая доля заказов от повторных?
- Какую долю выручки они генерируют?

### Задание 2.5 — Промокоды
- Средний чек с промокодом и без
- Возвращаются ли клиенты после промокода?
- Выгодны ли промокоды бизнесу?

### Бонус (для быстрых):
- Топ-5 категорий по выручке и среднему чеку
- Анализ по дням недели

---

## 💡 Этап 3: Инсайты и решения (20 мин)

### 3.1. Три главных инсайта
Формат:
> «Мы обнаружили, что [факт], что означает [бизнес-вывод]»

### 3.2. Три конкретных решения
Для каждого:
- Что именно сделать
- Какую метрику улучшит
- Ожидаемый эффект

### 3.3. Приоритет
Какое решение внедрять **первым** и почему?

---

## 🎤 Этап 4: Мини-презентация (3-5 мин)

2. **1 мин:** Топ-3 находки (с цифрами!)
3. **1.5 мин:** Рекомендации
4. **30 сек:** Приоритет №1

## 💡 Подсказки (если застряли)

1. Начните со сводной таблицы по месяцам — это даст общую картину
2. `Вставка → Сводная таблица` — ваш лучший друг
3. Обратите внимание на **время доставки** — оно может быть ключом
4. Сравните районы — один из них сильно отличается
5. Помните из лекции: **средние могут врать!** Смотрите медиану
