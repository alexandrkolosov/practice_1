# 🍕 Практическое задание: «Аналитик ЕдаВезде»
## Визуализация данных — искусство рассказывать истории | 1.5 часа

**Курс:** Business Intelligence для менеджеров
**Лекция 3:** Визуализация данных
**Формат:** Групповая работа (3-4 человека)
**Инструменты:** Excel/Google Sheets + Yandex DataLens

---

## 📖 Легенда

Вы — аналитическая команда сети доставки еды **«ЕдаВезде»**. Сеть работает в 4 городах: Москва, Санкт-Петербург, Казань, Екатеринбург.

**Ситуация:**
- CEO просит подготовить **визуальный отчёт** для совета директоров
- Есть данные за 9 месяцев (апрель–декабрь 2024)
- Главная проблема: **рейтинг падает**, клиенты жалуются
- Нужно: найти причину, показать красиво, предложить решение

**Ключевое требование:** Каждый график должен проходить **чек-лист 7 вопросов** из лекции!

## 📊 ЧАСТЬ 1: Excel — Правильные графики 

### Задание 1.1 — Постройте 5 графиков 

Для каждого графика:
- [ ] Выберите правильный ТИП
- [ ] Напишите заголовок-ВЫВОД (не описание!)
- [ ] Уберите лишнее по принципу Тафти
- [ ] Проверьте: понятно ли за 5 секунд?

**График 1: Тренд** — Как менялось среднее время доставки по месяцам?
> Подсказка: линейный. Ищите момент, когда время резко выросло.

**График 2: Сравнение** — Средний чек по городам
> Подсказка: горизонтальный столбчатый. Отсортируйте от большего к меньшему.

**График 3: Доли** — Структура выручки по категориям
> Подсказка: 7 категорий — pie chart НЕ подходит! Используйте столбчатый.

**График 4: Корреляция** — Зависимость оценки от времени доставки
> Подсказка: точечный (scatter). Есть ли зависимость?

**График 5: Паттерн** — Распределение заказов по часам
> Подсказка: столбчатый. Когда пик? Когда провал?

## 🔧 ЧАСТЬ 2: DataLens — Story-дашборд (

### Подготовка 
1. Откройте **datalens.yandex.ru**
2. Создайте подключение → загрузите `edavezde-orders.csv`
3. Создайте датасет

### Задание 2.1 — Вкладка «Обзор» 
Создайте первую вкладку дашборда:

- **4 KPI-карточки:** Общая выручка, Средний чек, Кол-во заказов, Средний рейтинг
- **1 линейный график:** Выручка по месяцам
- **1 столбчатый:** Выручка по городам
- **Фильтр-селектор:** Город

### Задание 2.2 — Вкладка «Проблемы»

- **Линейный график:** Среднее время доставки по месяцам (покажите скачок!)
- **Линейный график:** Средний рейтинг по месяцам (покажите падение!)
- **Столбчатый:** Средняя скидка по каналам (покажите агрегатор!)
- **Заголовок-текст:** напишите вывод прямо на дашборде

### Задание 2.3 — Вкладка «Рекомендации» 

- **Столбчатый:** Маржинальность по категориям (вычисляемое поле!)
- **Тепловая карта:** Заказы по дням и часам
- **Текстовый блок:** 3 конкретные рекомендации

### Сборка 
- Объедините 3 вкладки в один дашборд
- Добавьте общий фильтр по городу
- Назовите: «ЕдаВезде: отчёт для CEO»

## 🎤 Презентация Data Story (10 мин)


1. **Контекст** (30 сек): «ЕдаВезде — сеть доставки, 4 города, 5500 заказов за 9 месяцев»
2. **Конфликт** (1 мин): «Рейтинг упал с 4.0 до 3.4. Почему?» → переключите на вкладку «Проблемы»
3. **Инсайт** (1 мин): «Время доставки выросло с 29 до 39 мин в октябре-декабре» → покажите график
4. **Действие** (1 мин): «3 рекомендации» → переключите на вкладку «Рекомендации»
